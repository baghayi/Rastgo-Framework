بخش های دیگر

این هم یه فایلیه و کلاس نیست و برای لود باقی کلاس ها و فایل ها استفاده می شه و در حقیقت نکته شروع برنامه همین جاست.

اول از همه- فایل bootStrap.php رو لود می کنه که مقادیر رو داشته باشه و بتونه از طریف کلاس Loader باقی کلاس ها رو لود بکنه

بعد به ترتیب کلای Registry و ErrorReporting و Loader و Request و DatabaseConfig و Router صدا زده می شن (ازشون شی ایحاد می شه / به غیر ای Registry که شی تو خودش ایحاد می شه) و اشیاء ایجاد شده تو کلاس Registry نگه داری می شن

کلاس Router و DatabaseConfig هم داخل try-catch قرار داره که بتونه خطاها (استثناها) رو نمایش بده (البته اگه رخ بده)
