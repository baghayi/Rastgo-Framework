بخش های دیگر

این فایل هم فایلیه که تنظیماتش برای هر پروژه ای ممکنه تعییر بکنه و البته یکی از ثابت هاش باید تغییر بکنه

ثابت ها

ثابت اولی SCRIPT_ROOT_FOLDER_NAME هستش که اگه پروژه داخل یه پوشه ای اجرا می شه باید اسم اون پوشه برای این ثابت وارد بشه

توجه کنید که اگه از دامین اصلی و یا سابدامین استفاده می کنید این تابت رو باید خالی بزارید و نیازی نیست پرش کنید (چیزی واردش کنید)

منظور از اینکه پوشه باشه اسم پوشه رو وارد کنید یعنی اینطوری:

http://test.com/project/
http://subdomain.test.com/project/

که برای هر دوی این موارد باید project رو در ثابت وارد کنید ولی اگر اینطوری باشه: (یعنی توپوشه نباشه)

http://test.com/
http://subdomain.test.com/

نیازی نیست تو این ثابت جیزی بنویسید و اگه چیزی هم باشه مقدارش رو خالی کنید: اینطوری ''

که البته بطور پیش فرض نام فریمورک برای این ثابت تعریف شده: RastgoFramework

ثابت بعدی هم WEB_PROTOCOL هستش و پورتکل پیشفرض رو تعریف میکنید

مثل اگه از http استفاده می کنید می نویسید http و برای https و ... هم همون رو می نویسید که بوطر پیش فرض هم http هست

ثابت بعدی هم TEMPLATE_FOLDER_NAME هستش که نام پوشه قالب تون رو می نویسید که قالب پیش فرض اسمش defaultTemplate هست و همین مقدار بطور پیش فرض به این ثابت داده شده

که از این ثابت برای تغییر قالب سایت می شه استفاده کرد

ثابت Port_Number_In_URL برای این هستش که شماره پورت رو به آخر آدرس مون اضافه کنه . مثلا وقتی از IP بجای آدرس دامین استفاده می کنید و می خوایید شماره پورت هم تو آدرس قراره بگیره می تونید این ثابت رو True تعیین کنید و در غیر اینصورت False.
