هسته فریمورک

بخش های دیگر

فایل های هسته فریمورک جزء اصلی ترین فایل های فریمورک به شمار می یان که باعث می شن فریمورک روی پاهاش بایسته و دیگر بخش های فریمورک رو به نوعی به هم مرتبط کنن.

و طبیعتا هر چقدر قدرت و انعطاف پذیری این فایل ها در کنترل فریمورک بیشتر باشن دست توسعه دهنده هم همان مقدار برای گسترش برنامه هم باز تر خواهد بود.

فایل های هسته فریم ورک در پوشه core در شاخه اصلی فریمورک قرار داره.

فهرست فایل ها

  • baseController.php (فایل اصلی بخش Controller)
  • baseModel.php (فایل اصلی بخش Model)
  • bootStrap.php (تنظیمات ابتدایی فریمورک)
  • Loader.php (برای لودفایل ها توسط Namecpase شون)
  • Registry.php (پخش کننده شئ دیگر کلاس ها)
  • Request.php (برای کنترل URL)
  • Router.php (برای لود کنترلرهای درخواست شده)
  • View.php (فایل اصلی بخش View)
