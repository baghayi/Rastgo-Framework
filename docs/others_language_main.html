این پوشه محل نگهداری فایل های زبان هست و در مورد ساختارش با هم صحبت می کنیم.

همه فایل های زبان در پوشه ای قرار گرفته و در پوشه languages قرار می گیرن

مثلا پوشه persian یا english یا ... که از این اسم هم برای تعیین پوشه زبان در متد پابلیک setDefaultLang کلاس Translator استفاده می شه

تو پوشه ی مثلا english یا ... فایل های زبان رو قرار می دید که چون پسوند فایل زبان بصورت پیش فرض .php ست شده پسوند فایل ها رو هم باید .php تعیین بشه که البته این مورد توسط متد setLangFileExtension داخل کلاس Translator قابل تغییر هست

مثلا فایل index.php یا main.php یا common.php و یا ...

داخل فایل زبان مثلا main.php یا ... باید آرایه ای داشته باشید که مقادیر رو تو خودش ذخیره کرده باشه و در نهایت آرایه توسط return باید بازگشت داده بشه

و متونتون رو داخل آرایه ست کرده و برای هر کدام index ای تعریف می کنید و باقی کارها هم توسط کلاس Translator انجام خواهد شد.
