بخش های دیگر

کلاس baseModel یه کلاس abstract هست و همونطوری که از اسمش پیداست ازش در model هامون می تونیم ارث بری کنیم (البته اختیاری هست / ولی خوب طبیعتا از model هامون برای ارتباط با دیتابیس استفاده می شه و در نتیجه از baseModel هم باید ارث بری کرد)

همینطور این کلاس baseModel از کلاس PDO ارث بری می کنه و بخاطر همین بهتره از دستورات PDO برای ارتباط به دیتابیس استفاده بشه


متدها

متد اولی این کلاس متد سازنده هستش که وظیفه ارتباط با دیتابیسمون رو داره و مقادیر (اطلاعات) دیتابیسمون رو با استفاده از متدهای کلاس DatabaseConfig (که از طریق پروپرتی db از کلاس Registry می شه به شی کلاس DatabaseConfig دسترسی داشت ) می گیره و در نهایت مقادیر رو به متد سازنده پدر، یعنی PDO می فرسته و ارتباط با دیتابیس برقرار می شه.

این متد سازنده یه پارامتر داره و مقدار پیش فرضش FALSE هست و در صورتی که مقدار TRUE بگیره کوئری SET NAMES utf8 رو اجرا می کنه که برای کار با زبان هایی مثل فارسی نیازه که مقادیر به درستی در دیتابیس ذخیره بشن

متد دیگش هم که index هست و abstract و باید در model هایی که از این کلاس ارث بری می کنن این متد رو حتما تعریف کنن
