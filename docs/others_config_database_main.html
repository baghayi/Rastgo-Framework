بخش های دیگر

این فایل جهت قرار دادن تنظیمات مربوط به دیتابیس هست و در حقیقت دستورات دیتابیس اصلی مون که قراره باهاش بطور پیش فرض کار کنیم و در صورتی که حین کار بخواهیم به دیتابیس های دیگه ای کانکت بشیم می تونیم از طریق متدهای کلاس DatabaseConfig به این هدف دست یابیم.


اطلاعات دیتابیس باید بصورت آرایه در متغییر ای بنام $dbConfig ذخیره بشه.

کلید های آرایه شامل:

- name برای نام دیتابیس

- username نام کاربری دیتابیس

- password کلمه عبور دیتابیس

- host آدرس دیتابیسمون یا IP اش که معمولا یا localhost داده می شه و یا IP که این گزینه اختیاری هست و مقدار پیش فرض 127.0.0.1 هست.

- driver هم برای تعیین درایور دیتابیسمون هست که مثلا برای کار با mysql همین مقدار mysql رو بهش می دیم. که این گزینه اختیاری هست و مقدار پیش فرض mysql هست.

- port شماره پورت دیتابیس که وارد کردن این مقدار اختیاری هست و مقدار پیش فرض 3306 هست.

- pdoOptions برای تعیین آپشن های مربوط به کلاس PDO هست و این دستورات باید بصورت ارایه باشه و این اطلاعات در حقیقت به پارامتر چهارم کلاس PDO داده می شه. و همینطور این مقدار اختیاری هست و تعیین نکردنش مشکلی ایجاد نمی کنه.


آرایه بصورت استاتیک هستش و از همین آرایه در فایل index.php در ریشه برنامه مون ازش استفاده می شه و به کلاس DatabaseConfig تحویل داده می شه.
